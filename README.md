# Nourish

A daily food and wellness tracker built with React.

## Features

- Log meals by type (breakfast, lunch, dinner, snack)
- Track daily calorie intake with a visual progress ring
- Water intake tracker
- Mood logging
- Daily notes

## Tech Stack

- React
- Bootstrap
- Vite

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run start
```
